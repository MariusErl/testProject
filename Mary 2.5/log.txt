Version 2.3 to 2.3b
Change: Turned down amount of data communication between LFS-Mary (could give buffer-overflows for low-end PC's...)
Message given in console if using an unsupported LFS version with Mary
Stats and steal command updated after last TC-stats-page updates

Version 2.2b to 2.3
New: Mary-Minimap now sizeable, 3 different sizes. Add a value in the "minimapLoc=" setting, for example for using size 2: "minimapLoc=114,765,0.75,2"
Mary-Minimap drawn a lot more smoothly
Public version: Tracked car now highlighted on Mary-Minimap when you are not in chase. Disabled when you start/join a chase
Change: Chasebar in locator now only showing when you are actually chasing someone
New: Minimap now draws grid
New setting: minimapgrid=1	Enable/disable minimapgrid
New command: /o minimapgrid	Toggle minimapgrid on/off
